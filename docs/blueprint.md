# **App Name**: SpartaSync

## Core Features:

- Intelligent Shift Roster Tool: An AI-powered tool that automatically assigns personnel to event or fixed posts based on employee availability and specialty.
- Fixed Post Management: Dashboard to manage ongoing surveillance at permanent sites with specific security requirements.
- Event Scheduler: Interface for creating one-time event schedules with specialized roles and temporal demands.
- Staff Availability Database: A centralized database to track security agent profiles, contact information, and shift history.
- Scale Export & Print: Feature to export digital rosters into PDF formats for on-site display and compliance logging.
- Staff Portal: A simplified view for employees to log in and check their individual work scales on desktop or mobile browsers.

## Style Guidelines:

- Primary Color: Rich Gold (#D9B341) to evoke authority, excellence, and the 'Sparta' identity.
- Background Color: Deep Charcoal (#161411), which is a heavily desaturated and darkened version of the primary hue for a high-contrast dark mode feel.
- Accent Color: Burnished Sienna (#993D1A) providing a bold, contrasting alternative for critical status indicators and CTAs.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, high-tech, and disciplined appearance. Body font: 'Inter' (sans-serif) for maximum legibility in complex data tables.
- Linear, minimalist icons with a weighted border to mirror the strength of the security sector.
- A sidebar-based command center layout, providing easy navigation between Dashboard, Events, and Personnel records.
- Clean, instantaneous transitions between tabs to maintain the professional feel of a rapid-response tool.
